# 🧠 Realify – An Intelligent Fake News Detection System

Realify is a modern web-based platform that detects whether a news article is **REAL or FAKE** using Machine Learning.  
It combines a sleek interactive dashboard with an AI-powered backend to provide **instant credibility analysis and confidence scores**.

---

## ✨ Features

- 📰 **News Analysis** – Paste news text, headline, or URL for instant detection  
- 🤖 **AI Prediction** – Uses Machine Learning (Logistic Regression + TF-IDF)  
- 📊 **Confidence Score** – Shows how certain the model is  
- 📂 **History Tracking** – View previously analyzed results  
- 🔐 **User Authentication** – Login/Register system  
- 🎨 **Modern UI** – Animated dashboard with smooth UX  
- ⚡ **Fast Processing** – Real-time API-based predictions
